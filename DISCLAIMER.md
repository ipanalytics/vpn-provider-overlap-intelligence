# Disclaimer

This repository contains aggregate infrastructure analysis.

It does not publish raw VPN configuration files, full VPN IP feeds, credentials, private provider data, or bypass methods.

The data is provided for research, enrichment, and defensive analysis.

This is not a blocklist.

Presence in this dataset does not mean:
- the ASN is malicious;
- the provider is malicious;
- providers have common ownership;
- every IP in a prefix is a VPN;
- the data is complete or current forever.

Shared infrastructure can result from hosting providers, reseller platforms, white-label VPN services, migrations, public source artifacts, or stale records.

Use IP-level evidence, freshness, and source quality before making enforcement decisions.
